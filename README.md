# Green Campus Helper
A Java-based Command Line Interface (CLI) application designed to promote sustainability and track environmentally friendly activities within a campus environment. The system allows users to log green activities, track their environmental contributions, and enables administrators to monitor and manage participation.

This project demonstrates core Object-Oriented Programming (OOP) principles such as encapsulation, inheritance, abstraction, and polymorphism to create a modular and maintainable system.

---

# Features
## 👤 Participant Features
- Register and log in as a campus participant
- Record eco-friendly activities (recycling, tree planting, energy saving, etc.)
- Track sustainability contributions
- View activity history

  ## 🛠 Administrator Features
- Manage participant accounts
- Monitor logged sustainability activities
- Review overall campus sustainability progress
- Maintain system records
